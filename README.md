## Set up

1. Navigate to server

```
cd server
```

2. Install the node_modules using npm

```
npm install
```
3. Populate the Database

```
npm run seed
```
4. Start the redis server

```
redis-server
```

5. Run the server

```
npm start
```

# Endpoints

Access the product at:
http://localhost:4000/

# Set up

1. Navigate to server

```
cd server
```

2. Install the node_modules using npm

```
npm install
```

3. Run the server

```
npm start
```
4. Open another terminal and navigate to client

```
cd client
```

5. Install the node_modules using npm

```
npm install
```

6. Run the client

```
npm start
```
